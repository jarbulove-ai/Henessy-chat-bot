\# ☀️ WhatsApp Morning Fact Bot



Простой автономный чат-бот для WhatsApp, написанный на Python. Бот создан для личного использования (пет-проект) — он каждое утро автоматически отправляет в общий чат с друзьями теплое приветствие и случайный интересный факт дня.



\## 🚀 Как это работает

Скрипт запускается в облаке и работает в бесконечном цикле. Каждое утро в заданное время он:

1\. Делает запрос к бесплатному API и получает случайный факт на русском языке.

2\. Формирует красивое сообщение с использованием Markdown-разметки WhatsApp.

3\. Отправляет сообщение в целевую группу через стабильный облачный шлюз \*\*Green API\*\*.



\## 🛠 Технический стек

\* \*\*Язык:\*\* Python 3.10+

\* \*\*Библиотеки:\*\* `requests` (для работы с API)

\* \*\*API шлюз:\*\* Green API (бесплатный тариф \*Developer\*)

\* \*\*Хостинг:\*\* Render.com (развертывание в качестве \*Background Worker\*)



\## 📦 Структура проекта

\* `bot.py` — основной код бота с логикой проверки времени и отправки сообщений.

\* `requirements.txt` — список зависимостей для сборки проекта в облаке.

\* `README.md` — документация проекта.



\## ⚙️ Настройки и конфигурация



Бот использует переменные окружения (Environment Variables) для безопасного хранения ключей авторизации. Никакие токены не «зашиты» в код, что предотвращает их утечку при публикации репозитория.



Для работы скрипта используются следующие параметры:





| Переменная окружения | Значение | Описание |

| :--- | :--- | :--- |

| `GREEN\_API\_ID` | `7107646143` | ID вашего инстанса в Green API |

| `GREEN\_API\_TOKEN` | \*Ваш секретный токен\* | Токен авторизации инстанса |

| `GROUP\_CHAT\_ID` | `77023958782-1590737066@g.us` | Идентификатор группы \*\*Henessy\*\* |



\## 📝 Инструкция по локальному запуску



1\. Склонируйте репозиторий:

&#x20;  ```bash

&#x20;  git clone https://github.com

&#x20;  cd ИМЯ\_РЕПОЗИТОРИЯ

&#x20;  ```



2\. Установите зависимости:

&#x20;  ```bash

&#x20;  pip install -r requirements.txt

&#x20;  ```



3\. Задайте переменные окружения в вашей системе или временно подставьте их в код для теста, после чего запустите скрипт:

&#x20;  ```bash

&#x20;  python bot.py

&#x20;  ```



\## ☁️ Деплой в облако (Render)

1\. Создайте новый \*\*Background Worker\*\* на платформе \[Render](https://render.com).

2\. Подключите этот GitHub-репозиторий.

3\. Укажите команды для сборки и запуска:

&#x20;  \* \*\*Build Command:\*\* `pip install -r requirements.txt`

&#x20;  \* \*\*Start Command:\*\* `python bot.py`

4\. В разделе \*\*Advanced -> Environment Variables\*\* обязательно добавьте три переменные, указанные в таблице настроек выше.
